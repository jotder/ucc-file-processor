# Components

Reusable, independently-validated building blocks referenced by authored [flows](../flow-graph/design.md) via
`use:`. Stored as TOON files under the write-root registry and served over the `/components` + `/flows`
control routes.

# Concepts

* [Component registry](component-registry.md) - the store, the built-in types (grammar/schema/transform/sink/alert/connection), preview/test endpoints, and reference-safe deletion.
