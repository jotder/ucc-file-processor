## How to read this
- These docs **summarize and cross-link** the deep `docs/` reference material — they don't replace
  it. The `[KEEP-LIVE]` docs in the Archive Index remain the authoritative deep detail.
- Every document opens with a metadata block listing `sources_used`, `open_questions`, and
  `assumptions_made`.
- Where sources contradict, the conflict is **flagged** (see Conflict Report) rather than silently
  merged; genuinely open questions (e.g. console naming) are surfaced, not guessed.
